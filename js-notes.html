JavaScript क्या है? — Introduction

JavaScript web की "brain" है। HTML = skeleton, CSS = skin, JS = muscles + brain।

JS से website interactive बनती है — buttons click करना, forms validate करना, animations, API calls — सब JS से होता है!

  • Frontend + Backend दोनों के लिए (Node.js)
  • दुनिया की #1 भाषा — सबसे ज़्यादा use होने वाली
  • No compilation — Browser में directly चलती है
  • Versatile — Web, Mobile (React Native), Desktop (Electron), AI/ML भी!

Variables — Data Store करना

📦 Variables

Ch 04

Operators

➕ Operators
// Arithmetic
5 + 3   // 8  (जोड़)
10 - 3  // 7  (घटाव)
4 * 3   // 12 (गुणा)
10 / 3  // 3.33 (भाग)
10 % 3  // 1  (शेष - remainder)
2 ** 3  // 8  (power - 2³)

// Comparison — हमेशा === use करो!
5 === 5    // true  (strict equal ✅)
5 == "5"   // true  (loose equal ❌ avoid!)
5 !== 3    // true  (not equal)
5 > 3     // true
5 <= 5    // true

// Logical
true && true   // true  (AND - दोनों true)
true || false  // true  (OR - कोई एक true)
!true          // false (NOT - उल्टा)
Ch 05

Conditions — if else

🔀 Conditional Statements
const marks = 85;

if (marks >= 90) {
    console.log("A+ Grade 🌟");
} else if (marks >= 80) {
    console.log("A Grade 👍");
} else if (marks >= 60) {
    console.log("B Grade");
} else {
    console.log("Fail 😢");
}

// Ternary Operator (short if-else)
const result = marks >= 60 ? "Pass ✅" : "Fail ❌";

// Switch
const day = "Monday";
switch(day) {
    case "Monday": console.log("Start!"); break;
    case "Friday": console.log("Weekend!"); break;
    default: console.log("Normal day");
}

Loops — बार-बार दोहराना

🔄 Loops
// for loop — जब पता हो कितनी बार चलाना है
for (let i = 1; i <= 5; i++) {
    console.log(`Count: ${i}`);
}

// while loop — जब तक condition true हो
let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

// for...of — Array के items पर loop (⭐ most used)
const fruits = ["🍎", "🍌", "🍇"];
for (const fruit of fruits) {
    console.log(fruit);
}

// forEach — Array method
fruits.forEach((fruit, index) => {
    console.log(`${index}: ${fruit}`);
});
Ch 07

Functions — Code को Reusable बनाना

Function = एक machine जो input लेती है, कुछ करती है, और output देती है।

⚙️ Functions — Parameters & Return
// 1. Function Declaration
function add(a, b) {   // a, b = Parameters (input placeholders)
    return a + b;       // return = output (result वापस देना)
}

const sum = add(5, 3);  // 5, 3 = Arguments (actual values)
console.log(sum);       // 8

/*
 * 📖 समझो:
 * Parameters (a, b) = खाली boxes — function जब बनता है
 * Arguments (5, 3)  = actual values — function जब call होता है
 * Return            = result वापस करता है जहाँ से call हुआ
 * बिना return के function undefined देता है
 */

// 2. Arrow Function (modern way ⭐)
const multiply = (a, b) => a * b;
console.log(multiply(4, 5));  // 20

// 3. Default Parameters
function greet(name = "Guest") {
    return `Hello, ${name}!`;
}
console.log(greet());        // "Hello, Guest!"
console.log(greet("Anshu")); // "Hello, Anshu!"

🔑 Parameters vs Arguments:

• Parameter = function definition में — placeholder (a, b)

• Argument = function call में — actual value (5, 3)

• Return = function से value बाहर निकालता है — variable में store कर सकते हैं

Ch 08

Arrays — List of Items

📋 Array Methods
const nums = [10, 20, 30, 40, 50];

// Access
nums[0]          // 10 (index 0 से start)
nums.length      // 5

// Add/Remove
nums.push(60);    // End में add
nums.pop();       // End से remove
nums.unshift(0); // Start में add
nums.shift();     // Start से remove

// ⭐ Important Methods
const doubled = nums.map(n => n * 2);     // [20, 40, 60, 80, 100]
const big = nums.filter(n => n > 25);     // [30, 40, 50]
const total = nums.reduce((a,b) => a+b, 0); // 150
const found = nums.find(n => n > 25);     // 30 (first match)
const has = nums.includes(30);            // true

// Destructuring
const [first, second, ...rest] = nums;
// first=10, second=20, rest=[30,40,50]

Objects — Key-Value Data

🧱 Objects
const student = {
    name: "Anshu",
    age: 20,
    skills: ["HTML", "CSS", "JS"],
    greet() {
        return `Hi, I'm ${this.name}!`;
    }
};

// Access
student.name           // "Anshu"
student["age"]         // 20
student.greet()        // "Hi, I'm Anshu!"

// Destructuring
const { name, age } = student;

// Spread Operator
const updated = { ...student, age: 21 };

// Object methods
Object.keys(student)    // ["name", "age", "skills", "greet"]
Object.values(student)  // ["Anshu", 20, [...], ƒ]
Object.entries(student) // [["name","Anshu"], ...]

Async/Await — API Calls

🌐 Fetch API + Async/Await
// API से data लाना
async function getUsers() {
    try {
        const response = await fetch("https://jsonplaceholder.typicode.com/users");
        const users = await response.json();
        console.log(users);
    } catch (error) {
        console.log("Error:", error);
    }
}

getUsers();

/*
 * async  = function को asynchronous बनाता है
 * await  = promise resolve होने तक wait करो
 * try/catch = error handle करो
 * fetch  = HTTP request (API call)
 * .json() = response को JavaScript object में convert
 */

🎯 Interview Questions

Q1: let, const, var में फर्क?

→ var = function scoped, hoisted। let = block scoped, reassignable। const = block scoped, not reassignable।

Q2: == vs ===?

→ == type coercion करता है (5 == "5" true)। === strict check (type + value)। Always use ===!

Q3: Closure क्या है?

→ Inner function जो outer function के variables access कर सकती है, even after outer function return हो गया हो।

Q4: Event Bubbling vs Capturing?

→ Bubbling = child → parent (default)। Capturing = parent → child। e.stopPropagation() से रोक सकते हैं।

Q5: Promise vs Async/Await?

→ दोनों same काम करते हैं। Async/Await syntactically cleaner है — code synchronous जैसा दिखता है।

Q6: map() vs forEach()?

→ map() new array return करता है। forEach() कुछ return नहीं करता (side effects के लिए)।
