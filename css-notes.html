Ch 01

CSS क्या है? — Introduction

CSS = Cascading Style Sheets। यह HTML को सुंदर बनाती है — colors, fonts, spacing, layout सब CSS से आता है।

Simple शब्दों में: HTML = skeleton, CSS = skin + clothes। बिना CSS के हर website बोरिंग black text होगी!

CSS क्यों ज़रूरी है?

  • Design और Layout — page कैसा दिखेगा, कहाँ क्या होगा
  • Responsive Design — mobile/tablet/desktop पर सही दिखे
  • Animations — hover effects, transitions, smooth movements
  • Consistency — एक CSS file से पूरी site का look control
  • Performance — HTML में inline styles से CSS file बेहतर है
Ch 02

CSS कैसे जोड़ें? — 3 Methods

📎 Method 1: External CSS (Best ✅)
<!-- HTML file में -->
<link rel="stylesheet" href="style.css">

/* style.css file में */
body {
    background-color: #0a0a0f;
    color: white;
}
📎 Method 2: Internal CSS
<style>
    h1 { color: red; }
</style>
📎 Method 3: Inline CSS (Avoid ❌)
<p style="color: red; font-size: 20px;">Text</p>

⚡ Rule: हमेशा External CSS use करो! Code clean रहता है, caching होती है, और maintenance आसान।

Ch 03

Selectors — किसे Style दें?

🎯 CSS Selectors
/* 1. Element Selector */
p { color: white; }

/* 2. Class Selector (.) — multiple elements */
.highlight { background: yellow; }

/* 3. ID Selector (#) — unique element */
#header { font-size: 24px; }

/* 4. Descendant (अंदर वाला) */
nav a { color: blue; }

/* 5. Direct Child */
ul > li { list-style: none; }

/* 6. Pseudo-class */
a:hover { color: red; }
li:first-child { font-weight: bold; }
input:focus { border-color: blue; }

/* 7. Pseudo-element */
p::first-line { font-weight: bold; }
h1::before { content: "🔥 "; }

/* 8. Attribute Selector */
input[type="email"] { border: 2px solid green; }

/* 9. Universal Selector */
* { margin: 0; padding: 0; box-sizing: border-box; }

Specificity (कौन जीतेगा?)

Selector Specificity Priority
!important ∞ 🥇 सबसे ऊपर
Inline style 1000 🥈
ID #id 100 🥉
Class .class 10 4th
Element p 1 5th

Colors & Fonts

🎨 Colors & Typography
/* Color Formats */
.box {
    color: red;                    /* Named */

body {
    font-family: 'Inter', sans-serif;

    text-align: center;    /* left, right, justify */
    text-transform: uppercase; /* lowercase, capitalize */
    text-decoration: none; /* underline, line-through */
}
Ch 05

Box Model — हर element एक box है!

CSS में हर element एक box है। इस box के 4 parts हैं:

📦 Box Model
/*
  ┌─────────── Margin (बाहरी space) ────────────┐
  │  ┌──────── Border (किनारा) ──────────┐      │
  │  │  ┌──── Padding (अंदरी space) ───┐ │      │
  │  │  │                               │ │      │
  │  │  │     CONTENT (text/image)      │ │      │
  │  │  │                               │ │      │
  │  │  └───────────────────────────────┘ │      │
  │  └────────────────────────────────────┘      │
  └──────────────────────────────────────────────┘
*/

.box {
    width: 300px;
    padding: 20px;          /* Content के चारों ओर space */
    border: 2px solid #6c5ce7; /* Border */
    margin: 16px;           /* बाहरी space */
    border-radius: 12px;   /* Rounded corners */

    /* ⚡ IMPORTANT: box-sizing */
    box-sizing: border-box; /* width में padding+border include */
}

/* Shorthand: top right bottom left */
.box2 {
    margin: 10px 20px 10px 20px;  /* ↑ → ↓ ← */
    padding: 10px 20px;           /* ↕ ↔ */
}

⚠️ हमेशा करो: * { box-sizing: border-box; } — इसके बिना width calculation confusing हो जाती है!

Ch 06

Display & Position

📐 Display Property
.block    { display: block; }       /* पूरी width, new line (div, p, h1) */
.inline   { display: inline; }      /* Text के बीच, width/height नहीं (span, a) */
.ib       { display: inline-block; } /* Inline + width/height set कर सकते हैं */
.hidden   { display: none; }        /* पूरा छुपाओ */
.flex     { display: flex; }        /* Flexbox Layout */
.grid     { display: grid; }        /* Grid Layout */
📌 Position Property
.static   { position: static; }   /* Default — normal flow */
.relative { position: relative; top: 10px; } /* अपनी जगह से shift */
.absolute { position: absolute; top: 0; right: 0; } /* Nearest relative parent से */
.fixed    { position: fixed; top: 0; } /* Screen पर fix (navbar) */
.sticky   { position: sticky; top: 0; } /* Scroll तक normal, फिर fix */
Ch 07

Flexbox — 1D Layout King 👑

Flexbox items को एक line (row या column) में arrange करता है। Centering के लिए best!

🎯 Flexbox — Perfect Center
/* ⭐ सबसे ज़्यादा use होने वाला code */
.center-everything {
    display: flex;
    justify-content: center;  /* Horizontal ↔ */
    align-items: center;      /* Vertical ↕ */
    height: 100vh;            /* Full screen */
}

/* Navbar: logo left, menu right */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

/* Column layout (ऊपर-नीचे stack) */
.card {
    display: flex;
    flex-direction: column;
    gap: 16px;
}

/* Wrap — items next line पर जाएं */
.gallery {
    display: flex;
    flex-wrap: wrap;
    gap: 12px;
}
Property क्या करता है? Common Values
justify-content Main axis alignment center, space-between, space-evenly
align-items Cross axis alignment center, flex-start, flex-end, stretch
flex-direction Direction row, column, row-reverse
flex-wrap New line? nowrap, wrap
gap Items के बीच space 8px, 16px, 1rem
flex: 1 Child grows to fill flex-grow flex-shrink flex-basis
Ch 08

CSS Grid — 2D Layout King 👑

Grid items को rows AND columns दोनों में arrange करता है। Complex layouts के लिए best!

📊 Grid Layout
/* 3-column grid */
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;  /* 3 equal columns */
    gap: 20px;
}

/* Responsive auto-fit (magic!) */
.auto-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
}

/* Item spanning */
.hero-item {
    grid-column: 1 / 3;  /* 2 columns span */
    grid-row: 1 / 3;     /* 2 rows span */
}

Flexbox vs Grid: 1 direction (row OR column) = Flexbox। 2 directions (rows AND columns) = Grid। दोनों साथ भी use होते हैं!

Ch 09

Animations & Transitions

✨ Transitions (Smooth Changes)
.btn {
    background: #6c5ce7;
    transform: scale(1);
    transition: all 0.3s ease;  /* Smooth change */
}

.btn:hover {
    background: #a29bfe;
    transform: scale(1.05);      /* थोड़ा बड़ा */
    box-shadow: 0 8px 30px rgba(108,92,231,0.4);
}
🎬 Keyframe Animations
@keyframes fadeIn {
    from { opacity: 0; transform: translateY(20px); }
    to   { opacity: 1; transform: translateY(0); }
}

.card {
    animation: fadeIn 0.5s ease forwards;
}

@keyframes spin {
    0%   { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

.loader {
    animation: spin 1s linear infinite;
}
Ch 10

Responsive Design — हर Device पर Perfect

📱 Media Queries
/* Mobile First Approach — Default = Mobile */
.container {
    padding: 16px;
}

/* Tablet (768px+) */
@media (min-width: 768px) {
    .container { max-width: 720px; margin: 0 auto; }
    .grid { grid-template-columns: 1fr 1fr; }
}

/* Desktop (1024px+) */
@media (min-width: 1024px) {
    .container { max-width: 1100px; }
    .grid { grid-template-columns: 1fr 1fr 1fr; }
}

/* Responsive Units */
.text {
    font-size: clamp(1rem, 2.5vw, 2rem); /* Auto-scale! */
    width: min(90%, 1200px);
}
Ch 11

🤖 AI Tips for CSS

✅ AI से ये करवाएं:

• "इस layout को Flexbox/Grid से बनाओ"

• "Dark theme color palette suggest करो"

• "इस animation को smooth बनाओ"

• "इस CSS को responsive बनाओ"

⚠️ AI की common गलतियाँ:

• Overuse of !important — AI बहुत ज़्यादा !important लगा देता है

• Fixed px values — rem/em की जगह px use करता है

• Missing box-sizing — border-box भूल जाता है

• No mobile styles — responsive design miss कर सकता है

• Excessive nesting — selectors unnecessarily complex बना देता है

Ch 12

🎯 Interview Questions

Q1: Box Model explain करो।

→ Content + Padding + Border + Margin। box-sizing: border-box से width में padding/border include होता है।

Q2: display: none vs visibility: hidden?

→ none = element हटा देता है (space भी)। hidden = दिखता नहीं लेकिन space लेता है।

Q3: Flexbox vs Grid कब use करें?

→ 1D layout (row या column) = Flexbox। 2D layout (rows + columns) = Grid।

Q4: CSS Specificity order क्या है?

→ !important > inline > #id > .class > element > * (universal)।

Q5: em vs rem क्या है?

→ em = parent element के relative। rem = root (html) element के relative। Consistent sizing के लिए rem better है।
