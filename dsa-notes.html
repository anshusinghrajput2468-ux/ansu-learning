Ch 01

DSA क्या है? — Introduction

DSA = Data Structures + Algorithms। यह programming का सबसे important topic है — हर coding interview में पूछा जाता है!

  • Data Structure = data को organize करने का तरीका (Array, LinkedList, Tree...)
  • Algorithm = problem solve करने का step-by-step तरीका (Sort, Search...)
  • Interviews — Amazon, Google, Microsoft — सब DSA पूछते हैं
  • Problem Solving — logical thinking और optimize करने की skill
  • Better Code — efficient programs लिखने की ability

💡 Tip: DSA language-independent है! Concept एक बार समझ लो, किसी भी language (Java, Python, C++) में implement कर सकते हो।

Time & Space Complexity — Big O

Algorithm कितना fast है और कितनी memory लेता है — यह measure करती है Big O notation:

⏱️ Big O Notation
// O(1) — Constant (सबसे fast ⚡)
int first = arr[0];  // Direct access

// O(n) — Linear
for (int i = 0; i < n; i++) { }  // n बार चलेगा

// O(n²) — Quadratic (slow 🐌)
for (int i = 0; i < n; i++)
    for (int j = 0; j < n; j++) { } // n×n बार

// O(log n) — Logarithmic (fast! Binary Search)
// हर step में array आधा हो जाता है

// O(n log n) — Merge Sort, Quick Sort
Big O Name Example n=1000 operations
O(1) Constant Array access 1
O(log n) Logarithmic Binary Search ~10
O(n) Linear Linear Search 1,000
O(n log n) Linearithmic Merge Sort ~10,000
O(n²) Quadratic Bubble Sort 1,000,000
O(2ⁿ) Exponential Recursion (Fibonacci) 💀
Ch 03

Arrays — सबसे Basic Data Structure

📋 Array Operations (Java)
// Create Array
int[] nums = {10, 20, 30, 40, 50};

// Access → O(1)
nums[2]  // 30

// Linear Search → O(n)
int linearSearch(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) return i;
    }
    return -1;  // Not found
}

// Reverse Array → O(n)
void reverse(int[] arr) {
    int left = 0, right = arr.length - 1;
    while (left < right) {
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        left++; right--;
    }
}

// Two Pointer Technique — बहुत important! ⭐
// Find pair with given sum in sorted array
boolean pairSum(int[] arr, int target) {
    int l = 0, r = arr.length - 1;
    while (l < r) {
        int sum = arr[l] + arr[r];
        if (sum == target) return true;
        else if (sum < target) l++;
        else r--;
    }
    return false;
}
Ch 04

Strings — Text as Data

🔤 String Problems
// Palindrome Check — "madam" ↔ "madam"
boolean isPalindrome(String s) {
    int l = 0, r = s.length() - 1;
    while (l < r) {
        if (s.charAt(l) != s.charAt(r)) return false;
        l++; r--;
    }
    return true;
}

// Character Frequency Count
void charCount(String s) {
    HashMap<Character, Integer> map = new HashMap<>();
    for (char c : s.toCharArray()) {
        map.put(c, map.getOrDefault(c, 0) + 1);
    }
}
Ch 05

Linked List

Linked List = chain of nodes। हर node में data + next node का address होता है।

🔗 Singly Linked List
class Node {
    int data;
    Node next;
    Node(int d) { data = d; next = null; }
}

class LinkedList {
    Node head;

    // Add at beginning — O(1)
    void addFirst(int data) {
        Node newNode = new Node(data);
        newNode.next = head;
        head = newNode;
    }

    // Print all — O(n)
    void print() {
        Node temp = head;
        while (temp != null) {
            System.out.print(temp.data + " → ");
            temp = temp.next;
        }
        System.out.println("null");
    }
}
Operation Array Linked List
Access by index O(1) ✅ O(n) ❌
Insert at head O(n) ❌ O(1) ✅
Insert at end O(1) O(n) or O(1)*
Search O(n) O(n)
Memory Contiguous Scattered + extra pointer
Ch 06

Stack & Queue

📚 Stack (LIFO) & Queue (FIFO)
// Stack — Last In, First Out (थाली उल्टा रखना)
// push(add), pop(remove), peek(top देखो)
Stack<Integer> stack = new Stack<>();
stack.push(10);
stack.push(20);
stack.push(30);
stack.pop();   // 30 (last in, first out)
stack.peek();  // 20 (top element)

// Queue — First In, First Out (line में खड़ा होना)
// offer(add), poll(remove), peek(front देखो)
Queue<Integer> queue = new LinkedList<>();
queue.offer(10);
queue.offer(20);
queue.offer(30);
queue.poll();   // 10 (first in, first out)

// ⭐ Stack use: Balanced Parentheses Check
boolean isValid(String s) {
    Stack<Character> st = new Stack<>();
    for (char c : s.toCharArray()) {
        if (c == '(') st.push(')');
        else if (st.isEmpty() || st.pop() != c) return false;
    }
    return st.isEmpty();
}
Ch 07

Recursion — Function खुद को Call करना

🔄 Recursion Examples
// Factorial: n! = n × (n-1) × ... × 1
int factorial(int n) {
    if (n <= 1) return 1;  // Base case (रुको!)
    return n * factorial(n - 1); // Recursive call
}
// factorial(5) = 5×4×3×2×1 = 120

// Fibonacci: 0, 1, 1, 2, 3, 5, 8, 13...
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}
// ⚠️ O(2ⁿ) — very slow! Use DP to optimize

// Power: x^n
int power(int x, int n) {
    if (n == 0) return 1;
    return x * power(x, n-1);
}

⚠️ Recursion Rules: 1) हमेशा base case रखो (रुकने की condition)। 2) हर call में problem छोटी होनी चाहिए। 3) बिना base case = infinite loop = Stack Overflow!

Sorting Algorithms

🔀 Bubble Sort — O(n²)
void bubbleSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // Swap
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}
Algorithm Best Average Worst Stable?
Bubble Sort O(n) O(n²) O(n²) ✅
Selection Sort O(n²) O(n²) O(n²) ❌
Insertion Sort O(n) O(n²) O(n²) ✅
Merge Sort O(n log n) O(n log n) O(n log n) ✅
Quick Sort O(n log n) O(n log n) O(n²) ❌

Searching — Binary Search

🔍 Binary Search — O(log n)
// ⭐ Sorted array ज़रूरी है!
int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

/*  Array: [2, 5, 8, 12, 16, 23, 38, 56]
 *  Target: 23
 *  Step 1: mid=12, 23>12 → left half छोड़ो
 *  Step 2: mid=23, Found! ✅
 *  सिर्फ 2 steps में 8 elements से search हुआ!
 */

Trees — Binary Tree & BST

🌳 Binary Search Tree (BST)
/*     8        → Root
 *    / \
 *   3   10      → Left child < Parent < Right child
 *  / \    \
 * 1   6   14
 */

class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int v) { val = v; }
}

// Inorder Traversal: Left → Root → Right (sorted output!)
void inorder(TreeNode root) {
    if (root == null) return;
    inorder(root.left);
    System.out.print(root.val + " ");
    inorder(root.right);
}
// Output: 1 3 6 8 10 14 (sorted!)

🤖 AI Tips for DSA

✅ AI से ये करवाएं:

• "इस problem का approach step by step बताओ (code मत दो)"

• "मेरे solution की time complexity analyze करो"

• "इस brute-force solution को optimize करो"

⚠️ AI की common गलतियाँ:

• Direct solution दे देता है — पहले खुद try करो!

• Edge cases miss — empty array, single element, negative numbers

• Non-optimal solution — O(n²) दे सकता है जब O(n) possible हो

Ch 12

🎯 Interview Questions

Q1: Array vs Linked List कब use करें?

→ Random access → Array. Frequent insert/delete → LinkedList.

Q2: Stack vs Queue difference?

→ Stack = LIFO (plate stack). Queue = FIFO (bank line).

Q3: Binary Search की condition?

→ Array sorted होना चाहिए! Time: O(log n).

Q4: Stable vs Unstable sorting?

→ Stable = equal elements का original order maintain होता है (Merge Sort). Unstable = नहीं (Quick Sort).

Q5: HashMap vs TreeMap?

→ HashMap = O(1) access, unordered. TreeMap = O(log n) access, sorted keys.
